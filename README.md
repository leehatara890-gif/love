El Cupido Adivinador
====================

Página estática con un minijuego para adivinar un número (1-14).

Archivos principales:
- `index.html`
- `styles.css`
- `script.js`

Música de fondo de ejemplo:
- `assets/song.mp3` (descargada desde SoundHelix ejemplos para demo)

Uso local:
```bash
cd /workspaces/love
python3 -m http.server 8001
# abrir http://localhost:8001
```

Nota sobre la música:
- La pista incluida es para demo y pruebas (SoundHelix examples). Si quieres usar tu canción, reemplaza `assets/song.mp3` por tu archivo `.mp3` y haz commit.
